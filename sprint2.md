# Sprint 2: Password & Profile

Goal: user self-service features.

## 👤 Profile Features
- [x] Profile view (GET /api/users/profile)
- [x] Profile edit (PATCH /api/users/profile)
- [x] Upload profile image (POST /api/users/avatar)
- [x] View last login

## 🔐 Password & Security
- [x] Change password (Authenticated)
- [x] Forgot password (Generates reset token)
- [x] Reset password (Verifies token and updates password)
- [x] Email reset flow (Integration with mail service)

## 📅 Target Deliverable
- A secure self-service backend for profile management and account recovery.
